# @amlplugins/google-cloud-kms

Native `@google-cloud/kms` SDK re-exported under the `@amlplugins` namespace.

> Cloud KMS.

## Install

```bash
npm install @amlplugins/google-cloud-kms
```

## Use

```ts
import { /* named exports from @google-cloud/kms */ } from "@amlplugins/google-cloud-kms";
// or
import sdk from "@amlplugins/google-cloud-kms";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
